# Tic Tac Toe

Welcome to the Tic Tac Toe game repository! This classic game has been implemented in HTML, CSS, and JavaScript to provide a simple yet fun experience directly from your web browser.

## Overview

This Tic Tac Toe game allows two players to take turns marking spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Features

- A clean and intuitive user interface.
- Dynamic message display to announce the winner or a tie.
- A restart button to reset the game after it ends.

## Installation

No installation is required! You can play the game by simply opening the `index.html` file in any modern web browser.

## Usage

The game is straightforward to use:
1. Open the `index.html` file in a web browser.
2. Click on any cell to make a move. Players will alternate turns, starting with Player X.
3. The game will announce the winner or declare a tie if all cells are filled without a winning condition.
4. Click the "Restart" button to start a new game.
